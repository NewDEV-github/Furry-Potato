Introduction

Welcome to the Furry Potato documentation! This documentation is for the Furry Potato game.

About

This game is about DJ who wants to build up his career as a DJ.

Installation

Step 1: Cloning the repository

First, you need to clone the repository. You can do this by using the following command:

git clone git@github.com:NewDEV-github/Furry-Potato.git

One thing left to do is to download the NewDEVSharp library from download page. Once downloaded, place It's contents somewhere in the project's directory or whereever you want. Just remember to change the paths for files in the .csproj file.

Step 2: Installing Godot 4.0 Mono with .NET Framework 7.0

Next, you need to install Godot. You can do this by going to the Godot website and downloading the Godot Engine 4.0.3 Mono. Don't forget to select the Mono version and download .NET Framework 7.0.

Step 3: Opening the project

Finally, you need to open the project in Godot from it's project manager.
